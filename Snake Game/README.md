# 🐍 Snake Game

A classic Snake Game built using **HTML5 Canvas, CSS, and JavaScript**.

Control the snake, collect food, grow longer, and try to achieve the highest score without colliding with the walls or yourself.

---

## 🎮 Gameplay

* Use the arrow keys to control the snake.
* Eat food to increase your score.
* Each food item makes the snake grow longer.
* Avoid hitting the walls.
* Avoid colliding with your own body.
* The game ends when a collision occurs.

---

## 🚀 Features

* Responsive game canvas
* Real-time score tracking
* Random food generation
* Snake growth mechanics
* Wall collision detection
* Self-collision detection
* Restart game functionality
